# AI201 Project 4 - Provenance Guard

## Overview

Provenance Guard is an AI-powered text analysis system that estimates the likelihood that a piece of writing was generated by AI.

The system combines multiple independent detection signals to produce a confidence score between 0.0 and 1.0 while maintaining transparency through structured audit logging.

## Features

- LLM-as-Judge classification
- Stylometric analysis
- Confidence scoring
- Audit logging
- Flask API
- Transparency labels

## Technologies

- Python
- Flask
- Groq API
- Llama 3
- JSON
- Pandas
